## Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/rashmi-star/MedGuide.git
cd MedGuide
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```
The app will be available at `http://localhost:5173` (or next available port)
